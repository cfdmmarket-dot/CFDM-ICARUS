Entregue. Aqui está o que você recebeu:
📦 Pacote JARVIS — 6 arquivos
planner.py — O cérebro. Recebe o comando, chama a Claude API com um System Prompt de orquestração, analisa intenção, classifica complexidade (simple / moderate / complex), monta um plano com subtasks e dependências, e executa em ondas paralelas.
agents.py — Os 6 agentes especializados (ResearchAgent, CodeAgent, DatabaseAgent, FilesAgent, MemoryAgent, NotifierAgent) + o AgentRouter que mapeia qual agente chamar. Cada um tem seu próprio System Prompt e roda assíncrono.
voice.py — Interface de voz completa. STT via Whisper local (offline, privado, roda no seu Ubuntu). TTS via Coqui (local) ou ElevenLabs (cloud).
main.py — Entry point com 3 modos: texto interativo, voz, e comando direto via CLI.
requirements.txt — Todas as dependências comentadas e organizadas por módulo.
README.md — Setup completo para o seu Ubuntu.
