Microfone — Solução de Problemas

                

                    1. Use Chrome ou Chromium — Firefox não suporta Web Speech API
                

                

                    2. Acesse via http://localhost:8001 — HTTPS não é necessário em localhost
                

                

                    3. Quando clicar no 🎤, o Chrome pedirá permissão — clique Permitir
                

                

                    4. Se o mic não funcionar, vá em ⚙ Config → Testar mic para diagnóstico
                

                

                    5. No Linux, verifique: PulseAudio/PipeWire ativo e mic não mudo no sistema
